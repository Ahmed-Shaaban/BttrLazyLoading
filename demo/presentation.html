BttrLazyLoading

A responsive lazy loading JQuery plugin for Bootstrap 3 (but not only..)

Plugin dependencies: BttrLazyLoading depends on jQuery (this means jQuery must be included before the plugin files).

A 2 secondes delay has been added to all images for the demo.

Global options

Global options are options that you can define for the whole page. It is most of the time used before any BttrLazyLoading instanciation.

Name type default description
delay integer 0
threshold integer 0
animation string "bounceIn"
event string "scroll"
placeholder string (image base64) "data:image/gif;base64..."
container string (CSS selector) window
retina boolean false
triggermanually boolean false Whether or not to trigger the first image load manually.
updatemanually boolean false Whether or not to trigger the image update (Window resize) manually.
onBeforeLoad function($image, Object inst) null Called when the datepicker is closed, whether or not a date is selected.
onAfterLoad function($image, Object inst) null Called when the datepicker is closed, whether or not a date is selected.
onError function($image, Object inst) null Called when the datepicker is closed, whether or not a date is selected.
Type: Integer

delay

Set the delay to 4 secondes:

$('#img-option-delay').bttrlazyloading({
	delay: 4000
});
Type: Integer

threshold

The image will be automatically triggered after you scrolled 500px down from the top of the image.

$('#img-option-threshold').bttrlazyloading({
	threshold: -500
});
Type: String

animation

Here we will use the 'slideInLeft' animation to show how to overwrite the default "bounceIn" animation. You have access to all of the following animation created by Animate.css

['flipInX', 'flipInY', 'fadeIn', 'fadeInUp', 'fadeInDown', 'fadeInLeft', 'fadeInRight', 'fadeInUpBig', 'fadeInDownBig', 'fadeInLeftBig', 'fadeInRightBig', 'slideInDown', 'slideInLeft', 'slideInRight', 'bounceIn', 'bounceInDown', 'bounceInUp', 'bounceInLeft', 'bounceInRight', 'rotateIn', 'rotateInDownLeft', 'rotateInDownRight', 'rotateInUpLeft', 'rotateInUpRight', 'lightSpeedIn', 'rollIn']
$('#img-option-animation').bttrlazyloading({
	animation: 'slideInLeft'
});

Global ranges

Name type default description
ranges object 
{
 'xs' : 767,
 'sm' : 768,
 'md' : 992,
 'lg' : 1200
 }

Ranges follows Bootstrap 3 grid sizes. xs for phones (<768px), sm for small devices Tablets (≥768px), md for medium devices Desktops (≥992px) and lg for large devices Desktops (≥1200px). It defines from the user screen when a certain size of an image is needed.

For most needs, the default values (Bootstrap 3 grid) is enough.

Global methods

Global methods affect all BttrLazyLoading instanciations, it is used to set a global behavior for the whole page.

Returns: bttrlazyloading Global

setRanges()

$(".selector").bttrlazyloading({
	ranges: {
		xs: 767,
		sm: 768,
		md: 992,
		lg: 1200
	}
});
Returns: bttrlazyloading Global

setOptions()

$.bttrlazyloading.setOptions({
	img: {...},
	retina: true,
	transition: 'fadeInUp',
	delay: 1000,
	event: 'click',
	container: 'document.body',
	threshold: 666,
	placeholder: 'test',
	onBeforeLoad: function($img, bttrLazyLoading) {
	},
	onAfterLoad: function($img, bttrLazyLoading) {
	},
	onError: function($img, bttrLazyLoading) {
	}
});

Options

BttrLazyLoading options are the same as the global options and can be set in two different ways. Using jQuery data on the image element or directly on the instanciation:

Set options via jQuery data

onBeforeLoad, onAfterLoad and onError functions are not settable via jQuery data
<img id="test"
	data-bttrlazyloading-xs-src="../demo/img/768x200.gif"
	data-bttrlazyloading-xs-width="768"
	data-bttrlazyloading-xs-height="200"
	data-bttrlazyloading-sm-src="../demo/img/345x250.gif"
	data-bttrlazyloading-sm-width="345"
	data-bttrlazyloading-sm-height="250"
	data-bttrlazyloading-md-src="../demo/img/455x350.gif"
	data-bttrlazyloading-md-width="455"
	data-bttrlazyloading-md-height="350"
	data-bttrlazyloading-lg-src="../demo/img/360x300.gif"
	data-bttrlazyloading-lg-width="360"
	data-bttrlazyloading-lg-height="300"
	data-bttrlazyloading-transition="rotatedIn"
	data-bttrlazyloading-retina="true"
	data-bttrlazyloading-delay="2000"
	data-bttrlazyloading-event="mouseover"
	data-bttrlazyloading-container="document.body"
	data-bttrlazyloading-threshold="500"
	data-bttrlazyloading-placeholder="data:image/png;base64, iVBORw0KGgoAAAANSUhEUgAAAAUAAAAFCAYAAACNbyblAAAAHElEQVQI12P4//8/w38GIAXDIBKE0DHxgljNBAAO9TXL0Y4OHwAAAABJRU5ErkJggg=="
/>
Please not that the following examples are equivalent:
<img id="test"
	data-bttrlazyloading-xs-src="../demo/img/768x200.gif"
	data-bttrlazyloading-xs-width="768"
	data-bttrlazyloading-xs-height="200"
/>
<img id="test"
	data-bttrlazyloading-xs='{"src": "../demo/img/768x200.gif", "width" : 768,  "height" : 200}'
/>

Set options on Instanciation

$("#test").bttrlazyloading({
	img: {
		xs: {
			src: "../demo/img/768x200.gif",
			width: 111,
			height: 222
		},
		sm: {
			src: "../demo/img/345x250.gif",
			width: 333,
			height: 444
		},
		md: {
			src: "../demo/img/455x350.gif",
			width: 555,
			height: 666
		},
		lg: {
			src: "../demo/img/360x300.gif",
			width: 777,
			height: 888
		}
	},
	retina: true,
	transition: 'fadeInUp',
	delay: 1000,
	event: 'click',
	container: 'document.body',
	threshold: 666,
	placeholder: 'test',
	onBeforeLoad: function($img, bttrLazyLoading) {
	},
	onAfterLoad: function($img, bttrLazyLoading) {
	},
	onError: function($img, bttrLazyLoading) {
	}
})
