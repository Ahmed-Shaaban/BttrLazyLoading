BttrLazyLoading

A JQuery plugin for Image Responsive Lazy Loading

Plugin dependencies: BttrLazyLoading depends on jQuery (this means jQuery must be included before the plugin files).

Options

Name type default description
ranges object {'xs' : 767, 'sm' : 768, 'md' : 992, 'lg' : 1200}
delay integer 0
threshold integer 0
transition string "bounceIn"
event string "scroll"
placeholder string (image base64) "data:image/gif;base64..."
container string (CSS selector) window
retinaEnabled boolean false
onBeforeLoad function($image, Object inst) null Called when the datepicker is closed, whether or not a date is selected.
onAfterLoad function($image, Object inst) null Called when the datepicker is closed, whether or not a date is selected.
onError function($image, Object inst) null Called when the datepicker is closed, whether or not a date is selected.
